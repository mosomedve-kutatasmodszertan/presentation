\begin{frame}{Limitations}
\begin{itemize}

    \item Automated factuality checks introduce measurement uncertainty.
    \item \textbf{Data constraints:} 
    \begin{itemize}

    \end{itemize}
    \item Balancing data inclusion vs.\ factual stability remains unresolved.
